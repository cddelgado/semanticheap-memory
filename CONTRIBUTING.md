# Contributing

Thanks for helping improve Semantic Heap Memory.

## Ground rules

- Keep the conceptual model centered on `save` and `retrieve`.
- Prefer clear, deterministic, testable logic over clever complexity.
- Keep dependencies lightweight and local-first.

## Local workflow

```bash
pip install -e .
pytest
```

## Pull requests

Please include:

- what changed
- why it changed
- tests added/updated
